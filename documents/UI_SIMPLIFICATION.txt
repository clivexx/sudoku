USER EXPERIENCE:
================

BEFORE (Confusing):
  1. Select number (1-9)
  2. Toggle mode button (Number/Note)
  3. Click cell
  → Hard to remember which mode you're in!

AFTER (Simple):
  1. Click NUMBER button → places numbers
  OR
  2. Click NOTE button → edits notes
  → Instantly clear what will happen!


BUTTON COLORS:
==============

Number buttons (default): Grey (#E0E0E0)
Selected number button:   RED (#FF6B6B) ← Stands out!

Note buttons (default):   Grey (#E0E0E0)
Selected note button:     GREEN (#4CAF50) ← Different color!

This makes it IMPOSSIBLE to confuse which mode you're in.


STATE MACHINE:
==============

NEUTRAL MODE (nothing selected):
  - selected_number = None
  - note_mode = False
  - No buttons highlighted
  - Clicking cell does nothing

NUMBER MODE:
  - selected_number = 1-9
  - note_mode = False
  - One NUMBER button RED
  - Clicking cell → places that number

NOTE MODE:
  - selected_number = 1-9
  - note_mode = True
  - One NOTE button GREEN
  - Clicking cell → toggles that candidate


TRANSITIONS:
============

Neutral → Number mode:   Click any NUMBER button
Neutral → Note mode:     Click any NOTE button
Number → Neutral:        Click same NUMBER button again
Note → Neutral:          Click same NOTE button again
Number → Note:           Click any NOTE button
Note → Number:           Click any NUMBER button
Any → Neutral:           Click "Clear" button
Any → Neutral:           Click "Validate" button


TESTING CHECKLIST:
==================

□ Click Number 5 → RED background on number 5
□ Click cell → places 5
□ Click Number 5 again → deselected (neutral)
□ Click Note 3 → GREEN background on note 3
□ Click cell → toggles candidate 3
□ Click Note 3 again → deselected (neutral)
□ Click Number 7 then Note 2 → Note 2 GREEN (switches modes)
□ Click "Clear" → everything deselected
□ Click "Validate" → everything deselected + errors shown
□ "Show Errors" checkbox works
□ Can never have both a NUMBER and NOTE button selected


BENEFITS:
=========

✓ Simpler mental model (no mode toggle)

✓ Harder to make mistakes

✓ Faster workflow (one click instead of two)
✓ Same functionality, better UX
